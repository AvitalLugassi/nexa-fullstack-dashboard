# Nexa Server – Mock REST API

This folder contains the mock backend built using JSON Server.

It simulates a REST API used by the Nexa frontend application.

---
##  Live API

https://nexa-fullstack-dashboard.onrender.com

---

##  Resources

The database includes relational entities such as:

* users
* todos
* posts
* comments
* albums
* photos

All data is stored locally in `db.json`.

---

## ⚙️ Running Locally

```
npm install
npm run json:server
```

Server runs on:

http://localhost:3000

---

## 🎯 Purpose

This mock server enables:

* Full CRUD simulation
* Relational data navigation
* Rapid frontend development
* Testing async workflows

It can later be replaced with a real backend implementation.
